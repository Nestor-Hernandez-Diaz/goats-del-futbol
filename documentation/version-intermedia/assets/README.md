# Estructura de Carpetas de Assets

Esta carpeta contiene todos los recursos multimedia utilizados en el proyecto GOATS Fútbol.

## Estructura

- **images/**: Imágenes generales del sitio (banners, logos, fondos, etc.)
- **icons/**: Iconos y favicons utilizados en el sitio
- **videos/**: Archivos de video para el sitio
- **players/**: Recursos específicos de cada jugador
  - **messi/**: Recursos de Lionel Messi
  - **ronaldo/**: Recursos de Cristiano Ronaldo
  - **neymar/**: Recursos de Neymar Jr.

## Especificaciones Generales

### Nomenclatura de Archivos
- Usar nombres en minúsculas
- Separar palabras con guiones (ej: `banner-principal.jpg`)
- Incluir categoría en el nombre (ej: `logo-goats.svg`)
- Evitar caracteres especiales y espacios

### Formatos Recomendados
- **Imágenes**: JPG (fotografías), PNG (transparencias), SVG (gráficos vectoriales)
- **Iconos**: SVG (preferido), PNG (alternativa)
- **Videos**: MP4 (H.264)

### Optimización
- Todas las imágenes deben estar optimizadas para web
- Tamaño máximo de archivo:
  - Imágenes: 500KB (excepto banners: 1MB)
  - Iconos: 50KB
  - Videos: 10MB (clips cortos)

## Uso

Al agregar nuevos recursos, asegúrate de seguir la estructura de carpetas establecida y las especificaciones detalladas en cada README para mantener una organización adecuada del proyecto.
