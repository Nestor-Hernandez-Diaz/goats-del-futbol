# Archivos de Video

Esta carpeta contiene archivos de video para el sitio web de GOATS Fútbol.

## Notas Importantes

- Los videos deben estar en formato MP4 optimizado para web
- Tamaño máximo recomendado: 10MB para videos cortos
- Para videos más largos, considerar usar plataformas externas como YouTube o Vimeo
- Resolución recomendada: 1280x720 (720p) o 1920x1080 (1080p)

## Ejemplos de Videos a Incluir

- Highlights de jugadas
- Entrevistas cortas
- Momentos históricos
- Animaciones de transición

## Estructura de Carpetas

Se recomienda organizar los videos en subcarpetas según su categoría:

```
/videos
  /highlights
  /interviews
  /background
```
